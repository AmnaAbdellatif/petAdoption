# Adoption Portal Documentation 
bellow is the documentation for the apis included in House Renting portal backend.

### General Information
Authentication mechanism: JWT
Token TTL: 1 Hour
Refresh token TTL: 30 days 

### Login 
Path : /api/login

type : POST

body : {

    "username": "",
    "password": ""

}

### Register
Path : /api/register 

type : POST

body : {

    "username": "",
    "password": "",
    "type": "",
    "nom": "",
    "prenom": "",
    "cin": ""

}
### JWT refresh

type : ANY

body : {

    "refresh_token": "",

}
